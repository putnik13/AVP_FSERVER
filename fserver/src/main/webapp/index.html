Server is up and running ..


Recordings list available by following url: /recordings

Current version of rest api:
  • /api/startRecording - POST, starts video recording
  • /api/stopRecording - POST, stops video recording
  • /api/addChapter - POST, adds video chapter tag
  • /api/startRedirect - POST, starts redirect incoming stream to specified URL
  • /api/stopRedirect - POST, stops redirect incoming stream
  • /api/startRecordingAndRedirect - POST, starts recording and redirect incoming stream to specified URL
  • /api/stopRecordingAndRedirect - POST, stops recording and redirect incoming stream
